## ✅ 1. Criar o package.json automaticamente

## No terminal:

npm init -y


## Isso cria um package.json básico com configurações padrão.

## ✅ 2. Criar o package-lock.json

# O arquivo package-lock.json é criado automaticamente quando você instala qualquer pacote.

# Exemplo:

npm install


# ou quando instala um pacote novo, tipo:

npm install react


Assim que você instalar qualquer dependência, o package-lock.json aparece sozinho.
